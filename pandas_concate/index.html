纵向拼贴 axis = 0

  1. axis控制拼贴的方向

    pd.concat([df1, df2], ignore_index=False,axis=0)
"""
    A   B    C   D    F
0   A0  B0   C0  D0  NaN
1   A1  B1   C1  D1  NaN
2   A2  B2   C2  D2  NaN
3   A3  B3   C3  D3  NaN
2  NaN  B2  NaN  D2   F2
3  NaN  B3  NaN  D3   F3
4  NaN  B6  NaN  D6   F6
5  NaN  B7  NaN  D7   F7
"""

  2. ignore_index相当于合并后进行reset_index(), 重新生成索引

    pd.concat([df1, df2], ignore_index=True,axis=0)
"""
    A   B    C   D    F
0   A0  B0   C0  D0  NaN
1   A1  B1   C1  D1  NaN
2   A2  B2   C2  D2  NaN
3   A3  B3   C3  D3  NaN
4  NaN  B2  NaN  D2   F2
5  NaN  B3  NaN  D3   F3
6  NaN  B6  NaN  D6   F6
7  NaN  B7  NaN  D7   F7
"""

  3. join控制拼贴的方式,和merge一样

pd.concat([df1, df2], ignore_index=True,join="inner",axis=0)
"""
    B   D
0  B0  D0
1  B1  D1
2  B2  D2
3  B3  D3
4  B2  D2
5  B3  D3
6  B6  D6
7  B7  D7
"""
