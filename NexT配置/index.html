NexT配置

发表于 2021-05-14 更新于 2022-07-04

对next/_config.yml文件内容举例说明.

read_more_btn

用来控制当使用<!--more-->生成摘要时显示的read more按钮是否显示

read_more_btn: false

custom_file_path

用来配置自定义的js和样式等,详细参考官网

custom_file_path:
  #head: source/_data/head.njk
  #header: source/_data/header.njk
  #sidebar: source/_data/sidebar.njk
  #postMeta: source/_data/post-meta.njk
  #postBodyEnd: source/_data/post-body-end.njk
  #footer: source/_data/footer.njk
  #bodyEnd: source/_data/body-end.njk
  #variable: source/_data/variables.styl
  #mixin: source/_data/mixins.styl
  #style: source/_data/styles.styl

比如可以用来配置自定义的CSS样式

  • 开启自定义配置文件
    在next/_config.yml中开启自定义配置
custom_file_path:
  style: source/_data/styles.styl
  • 编辑自定义样式
    在hexo/source/_data/styles.styl中编辑
custom_file_path:
  style: source/_data/styles.styl

这里我自己的配置是:

.posts-expand .post-meta {
    margin: 3px 0 3px 0;
}
.post-block {
    padding: 20px;
}
p {
    margin: 0 0 0 0;
}
.post-body {
    font-size: 1em;
}
body {
    line-height: 1.5;
}

这种样式配合<!--more-->摘要,可以将首页文章展示做小,而不是展示文章所有内容.
