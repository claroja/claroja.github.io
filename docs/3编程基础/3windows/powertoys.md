## 热键映射

1. ctrl+g -> ctrl+h
2. ctrl+h -> left
3. ctrl+l -> right
4. ctrl+j -> down
5. ctrl+k -> Up
6. ctrl+y -> home
7. ctrl+o -> end
8. ctrl+i -> pageUp
9. ctrl+u -> pageDn


注意:
chome中:
1. ctrl+e 进入搜索框
2. f11 全屏显示
3. f12 开发者模式
