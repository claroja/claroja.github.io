# shortcuts

快捷键|描述
--|--
Ctrl + a | 全部选中
Ctrl + x | 剪切
Ctrl + c | 复制
Ctrl + v | 粘贴
Ctrl + z | 撤销操作
Ctrl + y | 撤销撤销的操作
Ctrl + d | 删除(shift+delete)
Ctrl + r | 刷新
Ctrl + w | 关闭窗口
Ctrl + n | 打开新窗口
Ctrl + Shift + n | 创建新文件夹
--|--
Alt + Tab | 切换应用
Alt + F4 | 关闭应用程序
Alt + ← | 后退
Alt + → | 前进
Alt + ↑ | 上一级目录
Alt + Enter | 选项的属性
--|--
F2 | 重命名选定项目
F5 | 刷新窗口
F6 | 循环浏览程序元素, shift可以逆向浏览
F11 | 最大化最小化窗口
--|--
Win + e | 打开资源管理器
Win + d | 显示桌面
Win + l | 锁屏
Win + g | 游戏相关
Win + r | 运行
Win + 方向键 | 1/4贴靠窗口
Win + Ctrl + d | 创建虚拟桌面
Win + Ctrl + 方向键 | 切换虚拟左面
Win + Ctrl + F4 | 关闭虚拟桌面
