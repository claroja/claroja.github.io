## Job 一次性任务
执行一次就停止
```yml
apiVersion: batch/v1
kind: Job
metadata:
  name: pi
spec:
  template:
    spec:
      containers:
      - name: pi
        image: perl
        command: ["perl",  "-Mbignum=bpi", "-wle", "print bpi(2000)"]
      restartPolicy: Never
  backoffLimit: 4
```

1. 创建`kubectl apply -f ./job.yaml`
2. 查看状态 `kubectl describe jobs/pi`
3. 查看输出 `kubectl logs $pods`



## cronjob 定时任务
cronjob.yml
```yml
apiVersion: batch/v1
kind: CronJob
metadata:
  name: hello
spec:
  schedule: "*/1 * * * *"
  jobTemplate:
    spec:
      template:
        spec:
          containers:
          - name: hello
            image: busybox
            imagePullPolicy: IfNotPresent
            command:
            - /bin/sh
            - -c
            - date; echo Hello from the Kubernetes cluster
          restartPolicy: OnFailure
```

Cron 时间表语法
```s
## ┌───────────── 分钟 (0 - 59)
## │ ┌───────────── 小时 (0 - 23)
## │ │ ┌───────────── 月的某天 (1 - 31)
## │ │ │ ┌───────────── 月份 (1 - 12)
## │ │ │ │ ┌───────────── 周的某天 (0 - 6) （周日到周一；在某些系统上，7 也是星期日）
## │ │ │ │ │                                   
## │ │ │ │ │
## │ │ │ │ │
## * * * * *
```
