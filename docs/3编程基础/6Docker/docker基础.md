# docker基础


1. 当我们利用Docker安装应用时，Docker会自动搜索镜像仓库并下载应用镜像（image）。镜像不仅包含应用本身，还包含应用运行所需要的环境、配置、系统函数库。
2. Docker会在运行镜像时创建一个隔离环境，称为容器（container）。
