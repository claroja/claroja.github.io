# 动画(animate)

```python
rect.animate({
  duration: 2000,
  delay: 1000,
  when: 'now',
  swing: true,
  times: 5,
  wait: 200
}).attr({ fill: '#f03' })
```

1. duration: 持续时间
2. delay: 等待多长时间开始动作
3. when: 
    1. now: 方法调用时执行
    2. absolute
    3. relative
    4. after
