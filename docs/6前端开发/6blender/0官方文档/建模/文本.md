# 文本

由于字体是矢量化的（由曲线构成），因此文本物体与曲线物体和曲面物体处于同一物体类型家族。


文本物体用于使用各种布局选项如左右对齐和文本框，来创建和渲染二维或三维文本。字符默认为平面填充的曲面，和闭合二维曲线一样。但是，跟曲线一样，可以对其进行挤出和使用修改器。
