# 属性(Properties)


## 变换(Transform)

1. 变换(Transform): 物体模式, 属性编辑器 ‣ 物体属性 ‣ 变换, 3D 视图 ‣ 侧栏 ‣ 变换

    侧栏区块中的 变换 面板，允许在 物体模式 中，查看和手动/数字控制物体的位置、旋转和其它属性。

    1. 位置(Position): 物体在局部坐标中的原点位置。
    2. 旋转(Rotation): 物体的局部方向，相对于全局轴及其自身的原点。
    3. 旋转模式(Rotation Mode)
        1. 欧拉(Euler):
        2. 坐标轴角度(Axis Angle):
        3. 四元数(Quaternion)
    4. 缩放(Scale)
    5. 维度(Dimensions): 体边界框的大小（与局部坐标轴对齐 —— 想象一个刚好足以容纳物体的纸板盒）
    6. 变换属性锁定(Transform Properties Locking)


2. 增量变换(Delta Transforms): 物体模式, 属性编辑器 ‣ 物体属性 ‣ 变换 ‣ 增量变换

    增量变换只是在上述变换之上应用的简单变换。


## 关系(Relation)

物体模式, 属性编辑器 ‣ 物体属性 ‣ 关系


1. 父级(Parent): 该物体是所选物体被设置父级的目标。
2. 父级类型(Parent Type):
3. 摄像机父级锁定(Camera Parent Lock): 当相机被锁定到视图，根父级将被变换，而不是相机。
4. 追踪轴(Tracking Axis): 指向 "前面" 方向的轴。
5. 向上轴(Up Axis): 指向 "向上" 方向的轴。
6. 通道编号(Pass Index): 定义物体在物体编号渲染通道中的编号。



## 集合

在 Blender 中，物体并不是直接属于场景的一部分。而是，它们都储存在主数据库里（一般来说就是 blend 文件）。当它们储存在场景中，它们将成为所谓的 场景集合 的一部分。所以根本上说，所有场景物体都属于这个特殊的集合。

1. 集合选项卡(Collections Tab): 属性 ‣ 集合属性

    集合属性选项卡允许方便地访问活动集合的属性。


    1. 限制(Restrictions)
        1. 可选(Selectable)
        2. 在渲染中禁用(Disable in Renders)
        3. 阻隔(Holdout)
        4. 仅间接(Indirect Only): 
    2. 实例化(Instancing)
        1. 实例偏移 X、Y、Z(Instance Offset X,Y,Z)
    3. 导出器(Exporters): 每个集合都可以导出为多种文件格式。
    4. 线条艺术(Line Art): 集合如何导入到线条艺术
    5. 自定义属性(Custom Properties)
2. 集合菜单(Collections Menu): 物体模式, 物体 ‣ 集合, M、Shift-M、Ctrl-G 等等。
    1. 移动到集合(Move to Collections): M
    2. 关联至集合(Link to Collection): Shift-M
    3. 创建新集合(Create New Collection): Ctrl-G
    4. 从集合中移除(Remove from Collection): Ctrl-Alt-G
    5. 从所有集合中移除(Remove from All Collections): Shift-Ctrl-Alt-G
    6. 将选中项添加到活动物体集合中(Add Selected to Active Objects Collection): Shift-Ctrl-G
    7. 从活动集合中移除所选(Remove Selected from Active Collection): Shift-Alt-G
3. 集合面板(Collections Panel): 物体模式, 物体选项卡 ‣ 集合

    物体已分配到的所有集合都列在“属性”的 物体选项卡 ‣ 集合面板 中。

    1. 添加到集合(Add to Collection): 
    2. 新建(New): 创建新集合并向其中添加所选物体。
    3. 名称(Name): 只需单击集合名称文本框，即可重命名该集合。
    4. X(Remove Collection):
    5. 其他菜单(Specials): 断开集合关联、选择集合、设置相对游标偏移量
    6. 偏移(Offset): 应用实例化集合与源物体的原点间的空间偏移量。


## 实例化(Instancing)

1. 顶点(Vertices): 物体模式, 属性编辑器 ‣ 物体属性 ‣ 实例化

    实例顶点 允许您在父对象的每个顶点的位置复制子对象。

2. 面(Faces): 物体模式, 属性编辑器 ‣ 物体属性 ‣ 实例化

    实例化面 是在父物体的每个面上复制物体的功能。

3. 集合(Collection): 物体模式, 属性 ‣ 物体属性 ‣ 实例化 ‣ 集合

    实例化集合 用于为每个其它物体的实例创建实例化集合。



## 可见性(Visibility): 

物体模式, 属性 ‣ 物体属性 ‣ 可见性

“可见性面板” 控制物体在视口和最终渲染中的交互方式。


## 视图显示(Viewport Display)

物体模式, 属性 ‣ 物体属性 ‣ 视图显示

配置 3D 视图的显示选项。

1. 显示(Show)
    1. 名称(Name): 显示物体名称
    2. 轴向(Axes): 表示物体的方向
    3. 线框(WireFrame): 在实体显示的上层显示物体的线框。
    4. 全部边(All Edges): 
    5. 纹理空间(Texture Space): 
    6. 阴影(Shadow):
    7. 在前面(In Front): 使物体显示在其他物体前面。
2. 显示为(Display As):
    1. 让您以较少的细节显示物体，从去除纹理到只显示边界框。
3. 颜色(Color):
    1. 物体在 线框 和 实体 视图着色模式下的颜色。
4. 边界(Bounds): 显示物体周围的边界形状。
