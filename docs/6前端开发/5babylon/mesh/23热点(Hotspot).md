
# [热点(hotspot)](https://doc.babylonjs.com/features/featuresDeepDive/mesh/hotspot/)










热点（Hotspot）是一项用于计算变形几何体三角形内世界空间位置的功能。例如，它可以计算由动画骨骼变形的网格上某个三角形的中心。然后，这个世界空间位置可以投影到屏幕上，创建一个 2D 热点。这使得动画网格的特定区域能更易被用户观察到。
