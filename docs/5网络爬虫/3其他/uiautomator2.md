# uiautomator2
uiautomator适用于adroid手机端自动化测试.而appium不仅适用于Android手机,同样适用于ios手机.
在Android系统场景中,uiautomator可跨app,在手机ROM层面实现交互.而appium只适合在应用内部,不能跨app.
而且appium可以实现webview界面的自动化测试,而uiautomator却不能.
