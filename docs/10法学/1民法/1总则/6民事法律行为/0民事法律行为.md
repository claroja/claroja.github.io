## 有效无效指的是条款, 不是整个合同
部分有效和部分无效, 多个条款合同, 可能部分条款有效, 部分条款无效, 此时不能说合同有效或无效





## 醉酒总结
1. 他人灌醉(利用) -> 显失公平 -> 可撤销 -> 受损害方
2. 自己喝醉
    1. 利用 -> 显失公平 -> 可撤销 -> 受损害方
    2. 未利用 -> 重大误解 -> 可撤销 -> 双方
