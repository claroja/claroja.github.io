# 逻辑

## 或与二选一

首先a或b=或者a或者b，他俩含义完全相同，记住一个即可。
a或b，至少有一个发生的意思，也就是对应3种结论：
1、a发生，b没发生
2、a没发生，b发生
3、a和b都发生

要么a要么b，是二选一的“要么”关系，对应两种结论：
1、a发生且b没发生
2、a没发生且b发生

在实际书写过程中, 建议格式如下:
A或B(二选一)
A或B(可多选)
