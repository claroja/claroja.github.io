# revealjs演示最佳实践


## 原生操作

1. 网页模式下F11全屏, 嵌入模式下F(Fullscreen)全屏
2. 在鼠标点击处放大缩小: Alt+LMB(Linux使用Ctrl+LMB). 需要开启zoom插件.
3. 幻灯片切换: 
    1. 切换动画+切换幻灯页: ←H, →L, ↑K, ↓J
    2. 切换幻灯页: Alt + ←/↑/→/↓
    3. 首位切换: Shift + ←/↑/→/↓
    4. 跳转到指定幻灯片: G(Go)
4. 幻灯片目录结构: ESC, O(Overview)
5. 打开演讲者模式: S
6. 快捷键帮助菜单: F1
7. 跳转到指定页面: G
