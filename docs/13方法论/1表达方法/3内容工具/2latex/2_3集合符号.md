# 集合符号


## 最佳实践

1. 集合里包含的是元素, 元素组合起来就是集合.
2. 集合与元素之间是`包含, 被包含`关系, 集合与集合之间是`子父等交`. 另外集合的运算是`交差并补`.
3. 概念也是集合的一种, 比如等腰三角形是三角形的一种, 可以表示为$等腰三角形 \subset 三角形$
