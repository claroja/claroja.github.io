# 沟通原理


## 沟通

沟通的三个层面:

1. 外在表达(行为语言): 既表达者的输出
2. 内在意思: 既表达者的内心想要表达的内容. 所谓词不达意说就是指这个.
3. 客观事实: 
    1. 同一概念的不同理解: 深度学习这个概念在从事算法专业的人员和普通人的概念是不一样的

## 表现

表现的三个层面:

1. 外在表达(行为语言): 既表演者的输出
2. 内在意思: 既表达者的内心想要表达的内容. 这个内容就是想让别人觉得自己是什么样子.
3. 客观事实: 自己本身的样子
