# 逻辑符号(Logic Notation)

## 最佳实践

1. AB有且只有一个为真， 如何用符号表示:
    1. 数学符号表示: $(A \land \neg B) \lor (\neg A \land B)$
    2. 编程符号表示: `(A &&!B) || (!A && B)`

2. 蕴含关系

    “蕴含关系”（通常用“→”表示）的特定符号。但是，在逻辑编程和条件判断中，可以通过条件语句来实现类似的逻辑。例如，在 C、C++、Java 等语言中，可以使用 if 语句来模拟蕴含关系。

## 数学逻辑运算(logic)

symbols|symbols|meaning|examples
--|--|--|--
`$\exists$`|$\exists$|There exists at least one|$\exists x:F(x)$ means that there is at least one x such that F(x) is true.
`$\exists!$`|$\exists!$|There exists one and only one|$\exists! x:F(x)$ means that there is exactly one x such that F(x) is true.
`$\nexists$`|$\nexists$|there is no|$\exists! x:F(x)$ means that there is no one x such that F(x) is true.
`$\forall$`|$\forall$|for all|$\forall n > 1; n^2 > 1$
`$\neg$`|$\neg$|not (logical not)|Statement A is true, $\neg A$ is false
`$\lor$`|$\lor$|or (logical or)|if A or B is true, $A \lor B = true$
`$\land$`|$\land$|and (logical and)| if A and B is true, $A \land B = true$
`$\Rightarrow$ $\rightarrow$`|$\Rightarrow$ $\rightarrow$|right implication|左蕴含右, 充分条件
`$\Leftarrow$ $\leftarrow$`|$\Leftarrow$ $\leftarrow$|left implication|右蕴含左, 必要条件
`$\Leftrightarrow$ $\leftrightarrow$`|$\Leftrightarrow$ $\leftrightarrow$|equivalence|左右相互蕴含, 充要条件
`$\nRightarrow$ $\nrightarrow$`|$\nRightarrow$ $\nrightarrow$|right implication|加n, 取反
`$\nLeftarrow$ $\nleftarrow$`|$\nLeftarrow$ $\nleftarrow$|left implication|加n, 取反
`$\nLeftrightarrow$ $\nleftrightarrow$`|$\nLeftrightarrow$ $\nleftrightarrow$|equivalence|加n, 取反
`$\Longrightarrow$ $\longrightarrow$`|$\Longrightarrow$ $\longrightarrow$|right implication|加long, 加长
`$\Longleftarrow$ $\longleftarrow$`|$\Longleftarrow$ $\longleftarrow$|left implication|加long, 加长
`$\Longleftrightarrow$ $\longleftrightarrow$`|$\Longleftrightarrow$ $\longleftrightarrow$|equivalence|加long, 加长

还有两个快捷便于记忆的语义符号:

symbols|symbols|meaning|examples
--|--|--|--
`$\implies$`|$\implies$|implies(蕴含)|$x = 2 \implies x^2 = 4$
`$\iff$`|$\iff$|if and only if, equivalence(等价)|$x +1 =y+1 \iff x =y$
