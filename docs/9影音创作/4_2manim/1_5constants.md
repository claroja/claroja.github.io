# [constants](https://docs.manim.community/en/stable/reference/manim.constants.html)


| 指令 | 坐标 (X, Y, Z) | 含义 |
| :--- | :--- | :--- |
| **ORIGIN** | `(0.0, 0.0, 0.0)` | 坐标系的中心。 |
| **UP** | `(0.0, 1.0, 0.0)` | 沿 Y 轴正方向移动一个单位。 |
| **DOWN** | `(0.0, -1.0, 0.0)` | 沿 Y 轴负方向移动一个单位。 |
| **RIGHT** | `(1.0, 0.0, 0.0)` | 沿 X 轴正方向移动一个单位。 |
| **LEFT** | `(-1.0, 0.0, 0.0)` | 沿 X 轴负方向移动一个单位。 |
| **IN** | `(0.0, 0.0, -1.0)` | 沿 Z 轴负方向移动一个单位。 |
| **OUT** | `(0.0, 0.0, 1.0)` | 沿 Z 轴正方向移动一个单位。 |
| **UL** | `(-1.0, 1.0, 0.0)` | 向上移动一个单位 + 向左移动一个单位。 |
| **UR** | `(1.0, 1.0, 0.0)` | 向上移动一个单位 + 向右移动一个单位。 |
| **DL** | `(-1.0, -1.0, 0.0)` | 向下移动一个单位 + 向左移动一个单位。 |
| **DR** | `(1.0, -1.0, 0.0)` | 向下移动一个单位 + 向右移动一个单位。 |
