# arc

arc用来制作圆弧


```python
arc=Arc(arc_center= ORIGN,  # 圆心位置
        radius=2.0,  # 半径
        stroke_width=20.0,  # 圆弧宽度
        start_angle=45*DEGREES,  # 圆弧开始的角度(从此开始画圆)
        angle=90*DEGREES,  # 圆弧的角度
        color=BLACK)  # 默认是白色

```


`arc=Arc(angle=TAU/3).add_tip(at_start=True)`: 给圆弧两端加箭头

## Annulus
圆环

```python
annulus = Annulus(
    outer_radius=3,  # 内径
    inner_radius=2.5,  # 外径, 外径-内径既为圆环的宽度
    fill_color=BLUE,
    fill_opacity=0.6,
    stroke_width=10,
    stroke_color=YELLOW,  # 圆环的内外两侧
)
```


## AnnularSector

```python
ann_sec = AnnularSector(

    stroke_color=YELLOW,
)
```
