

播放指示器(时间垂直线):

要对齐剪辑边缘和标记，请按住 Shift 键并在时间轴面板中拖动播放指示器。


对于键盘快捷键：

- 使用左或右箭头键移动一帧。
- 按住 Shift + 箭头键，一次移动五帧。



使用以下任何技巧放大和缩小序列。

使用+放大。
使用–缩小。


处理长序列时，某些剪辑可能不在视图中。要水平滚动并显示序列中隐藏的部分

- 使用Page Up键向左滚动。
- 按 Page Down 键向右滚动。



[序列设置参考](https://helpx.adobe.com/cn/premiere/desktop/edit-projects/change-clip-sequence/sequence-settings-reference.html)




移动剪辑

可以使用键盘方向键向左或向右微移选定的剪辑。这适用于精确调整。按住 Ctrl (Windows) 或 Command (macOS) 键的同时按方向键可进行更大的微移调整。

如果您希望在移动剪辑时其他剪辑自动调整以腾出空间，请使用波纹编辑工具。

使用逗号（,）和句号（.）键分别执行插入和覆盖编辑。这些快捷键允许您在播放头位置插入剪辑（逗号）或在播放头位置覆盖剪辑（句号）。


重新排列:  Ctrl + Alt（Windows）
