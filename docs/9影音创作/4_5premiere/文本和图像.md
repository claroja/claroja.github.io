## 绘制对象


钢笔工具绘制形状、创建遮罩和调整曲线，以实现精确的图形和选择性视觉效果。


使用钢笔工具绘图的最简单路径，是具有两个顶点的直线。

1. 按住您希望让直线路径起始的点，然后单击希望让该路径结束的点（不要拖动）。
2. 继续单击，以便为其他直线路径设置顶点。
3. 要绘制角，可在以下情况下按住 Shift：

    1. 添加点以绘制具有完美的 0°、45° 和 90° 角度的直线。
    2. 拖动现有顶点可使其沿 0°、45° 和 90° 角度移动。
    3. 拖动贝塞尔手柄可将其约束在 0°、45° 和 90° 角度。
    4. 按住 Shift + Opt/Alt 并拖动贝塞尔手柄，可将贝塞尔曲线拆分为两条，同时将您点击的手柄约束为 45° 增量移动。


绘制曲线: 通过调整锚点来创建曲线，从而形成平滑的曲线路径。

1. 选择钢笔工具  图标，然后按住您希望曲线开始的点。此时将显示一个顶点，并且钢笔工具指针将变为箭头。
2. 拖动以修改顶点两条方向线的长度和方向，然后释放鼠标按钮。
3. 将钢笔工具放置在您希望结束曲线路径的位置，然后尝试以下任一操作：

    要创建 C 形曲线，请沿与您拖动前一方向线的方向相反的方向进行拖动，然后释放鼠标按键。


钢笔、矩形、椭圆和多边形工具创建任意形状和路径。


1. 选择并按住矩形工具以查看椭圆和多边形工具。 
2. 选择一个形状工具，然后将其拖动到节目监视器上以绘制形状。
3. 使用选择工具 (V) 移动形状或更改其宽度、高度、旋转或锚点。
4. 使用属性面板中的选项来进一步优化形状。

    1. 外观：
        1. 填充：使用任何所需颜色填充形状。
        2. 描边：向形状添加外描边、内描边或者中心描边，或者添加多条描边。
        3. 阴影：向形状添加单独阴影或多重阴影以创建众多有趣的效果。
        4. 形状蒙版：创建一个充当蒙版的形状图层，仅显示形状内的底层视频或图形。
    2. 对齐和变换：按其顶部边缘、垂直中心、底部边缘、左边缘、水平中心或右边缘对齐对象。
    3. 响应式设计 – 位置：设计图形，使其自动适应视频帧长宽比的变化，或自动适应其他图形图1. 层的位置或缩放属性。


对齐

选择Windows > Properties。右键单击要对齐的文本或对象，然后从以下三个选项中选择：

- 对齐到视频帧：将每个对象单独对齐到Program Monitor。
- 作为组对齐到视频帧：将多个选定对象作为单个组对齐到Program Monitor。
- 对齐到选区：相对于关键（主要）对象对齐对象，例如左对齐时最左侧的对象或底部对齐时最低的对象。


分布对象

使用“分布”选项确保对象在垂直或水平方向上等距分布。

您可以使用“分布”选项确保对象在垂直或水平方向上等距分布。

- 在属性面板中选择对象。
- 在对齐和变换选项卡中，选择对齐选项。
- 使用对齐字段中的按钮，选择将对象相对于中心垂直或水平对齐，或者左对齐、右对齐、顶部对齐或底部对齐。


效果

固定效果是预先应用或内置于每个片段中的效果，包括 动作、不透明度、时间重映射和 音量调整。

- 运动: 制作动画并调整位置、缩放、旋转、防闪烁，或与其他剪辑合成。
- 不透明度: 降低剪辑的不透明度，用于叠加、淡入淡出和溶解等效果。
- 时间重映射: 修改播放速度、冻结帧，并创建慢动作或快动作效果。
- 音量: 针对剪辑的任何部分减速、加速或倒放或者将帧冻结。这可对这些变化的加速或减速进行精细控制。

创建蒙版


蒙版创建后即成为镂空模板，仅显示其下方的视频或图像部分。随后可在蒙版区域内应用效果或调整，而区域外内容保持不变。

您可以通过两种方法创建蒙版：

- 使用矩形工具、椭圆工具和多边形工具
- 使用钢笔工具绘制蒙版。

请在效果控制面板中选择椭圆、矩形或多边形工具来创建蒙版。




编辑蒙版的外观


- 要更改蒙版的形状，请在 Windows 上按 Alt 键或在 macOS 上按 Option 键，然后单击并拖动形状的顶点。
- 要调整蒙版大小，请将光标置于顶点外侧并按 Shift 键。当光标变为双箭头图标时，将其拖动至所需尺寸。
- 要旋转蒙版，请将光标置于顶点外侧。当光标变为弯曲的双箭头图标时，拖动至所需角度。
- 要更改蒙版的形状，请移动、添加或删除顶点，然后拖动它们，直到获得所需的蒙版形状。

    - 移动：使用选择工具拖动顶点。
    - 添加：将光标悬停在蒙版边缘，同时按住 Ctrl（Windows 系统）或 Cmd（macOS 系统）。当光标变为带有加号  图标的钢笔形状时，选择您想要添加顶点的位置。
    - 删除：将光标悬停在要删除的点上，同时按住 Ctrl（Windows 系统）或 Cmd（macOS 系统）。当光标变为带减号的笔形状时，选择该点。
- 要移动蒙版：

    - 要将蒙版移动一个单位，请选择一个控制点，然后使用键盘上的箭头键微调该点。
    - 要将蒙版移动五个单位，请选择一个控制点，然后按住 Shift 键的同时使用箭头键微调该点。
- 要删除蒙版，请在效果控制面板中选择蒙版，然后按键盘上的 DELETE 键。


跟踪蒙版

使用蒙版跟踪可以完成诸如模糊人脸、为移动对象添加文字或图形、隔离特定元素或对跟踪区域应用颜色校正等任务。为了实现准确跟踪，蒙版区域内需要有一个轮廓清晰、对比度高的对象或特征。

1. 打开效果控件面板，然后选择蒙版并展开其属性。
2. 在蒙版路径选项中，选择向前跟踪所选蒙版按钮（向右箭头），以在对象移动时开始跟踪蒙版。如有需要，您也可以向后跟踪。
3. 选择跟踪方法：

    1. 位置：仅逐帧跟踪蒙版位置。
    2. 位置和旋转：跟踪蒙版位置，并根据每帧的需要改变旋转角度。
    3. 位置、缩放和旋转：跟踪蒙版位置，并在帧移动时自动缩放和旋转蒙版。
