# 如何记忆

## 记忆的一般原则

- 是评估, 不是记原文, 要把采分点抠出来
- 反复是记忆之母
- 无中生有最难记(和已有的知识建立联系)
- 似是而非最徒劳(宁可坚决果断的错, 不可犹豫不决的对)
- 不强行要求理解记忆(死记硬背概念有时比理解记忆效率高)
- 以记关键语义块为准, 将记忆的语义块用自己的话还原
- 理清逻辑关系
    - 前后顺序关系, 比如步骤
    - 条件关系, 充分或必要
    - 集合关系, 包含或不包含
- 记忆桥梁🌰比如事前故意的因果关系认识错误, 分歧点在于介入因素有没有切断因果关系, 然后是推导出结论, 切断因果关系如何判断; 不切断因果关系如何判断.

    1. 事前故意的因果关系认识错误是考察点
    2. 介入因素有没有切断因果关系是记忆桥梁
    3. 结论, 不需要记忆, 是根据记忆桥梁推断出来的

    ✨同时这里也看出知识点与考察点的区别. 知识点介入因素的概念, 并不是考察点, 而是事前故意的因果关系认识错误这个考察点的一部分
    ✨顺口溜是一种记忆桥梁, 好的顺口溜可以简化记忆; 不好的顺口溜则会加重记忆负担


## 学习金字塔

1. 听讲: 听讲是效率最差的, 因为脑子要不间断的处理接收到的语音信号, 不能同时思考
2. 阅读: 相较于听讲好一些, 可以停下来思考
3. 视听: 相交于文字好一些, 更具体, 形象
4. 演示: 相较于图画好一些, 可以展示多个画面. 但成本大.
5. 现实交互: 相较于虚拟交互更好一些, 更真实. 但成本大.
6. 讨论: 有更多的思考
7. 实践: 自己上车实际操作
8. 教授: 给别人讲解(费曼学习法)





## 记忆方法实现

每篇文章:

1. 一个一级标题是题目
2. 第一个二级标题是考察点
    1. 第一个三级标题是: `考察问`, 将`考察点`中的拆分点抠出来, 使用`()`来代替. 放在第一是因为方便复习, 不然上来就看到考察点记忆效果就不好了.
    2. 第二个三级标题是: `考察点`, 根据`知识点`总结出来
    3. 第三个三级标题是: `题目总结`, 列表形式, 列表元素是题型名称, 列表内容是具体的题目
3. 第二个~第N个二级标题是书本内容
4. 最后一个二级标题是参考文献
