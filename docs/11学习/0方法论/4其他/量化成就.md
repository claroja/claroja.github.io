# 量化成就

一个人的成就(achievement)可以用3个维度评价:

- 影响人数(the number of influenced people)(N): 数量
- 给每个人带来的价值(The value brought to each person)(V): 客观带来的价值
- 反馈系数(feedback coefficient)(F): 每个人对该价值的评估

$$
\text{ACH.} = \sum_1^N VF
$$


## 个人他人的损益:

1. 损人损己的事情: 理性一点就不会做
2. 损人益己的事情: 道德一点就不会做
3. 益人益己的事情: 不好找
4. 益人损己的事情: 很高境界
