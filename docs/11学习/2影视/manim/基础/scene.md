# scene


`scene`包含了`frame`,`mobject`,`cameras`等一系列的对象.


方法|描述
--|--
construct|所有的操作都写在此处
add|添加`mobject`对象
bring_to_back|将`mobject`放在最后
bring_to_front|将`mobject`放在最前
clear|移除所有`mobject`
next_section|
pause|暂停
play|表演动画
remove|移除对象
wait|等待
