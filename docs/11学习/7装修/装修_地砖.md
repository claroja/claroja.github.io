# 地砖

## 最佳实践

1. 亮光
2. 米白色


## 地砖分类

根据瓷砖的表面反光程度可以分为:

1. 亮光
2. 哑光
3. 柔光


## 亮光砖

表面光滑

1. 采光好
    1. 适合采光不好的, 低楼层小户型
    2. 不适合做无主灯, 灯光打下去会有光斑
2. 防滑性不如哑光砖
3. 清洁容易, 适合懒人
4. 相对便宜


## 哑光砖

表面凹凸不平

1. 采光差
    1. 适合采光好的户型
    2. 适合无主灯, 无主灯
2. 耐磨防滑
3. 清洁难, 不适合懒人
4. 贵, 环氧彩砂做美缝, 才能达到见砖不见缝的效果

## 柔光转

表面像鹅卵石一样光滑

分类:

1. 肌肤釉: 清洁度高
2. 緞光: 清洁度高
3. 糖果釉: 清洁度中
4. 细干粒: 清洁度中
5. 柔抛砖: 清洁度低,
