# 复合类型

## 数组(array)

数组的类型声明, 两种方式:

1. 类型[]
2. Array<类型>

```typescript
let list: number[] = [1, 2, 3];
let list: Array<number> = [1, 2, 3];
```

## 元组(tuple)

元组就是固定长度的数组
语法：[类型, 类型, 类型]

```typescript
let x: [string, number];
x = ["hello", 10]; 
```

## 枚举(enum)

```typescript
enum Gender{
    Male,
    Female
}

let i: {name: string, gender: Gender};
i = {
    name: '孙悟空',
    gender: Gender.Male // 'male'
}
```
