# Containers


时间维度(时间复杂度):算法所消耗的时间。
空间维度(空间复杂度):算法需要占用多少内存空间。

## 时间复杂度测量
由于直接测试程序运行时间, 可能会受到外界因素影响, 比如CPU时间等
大O符号表示法, 即 T(n) = O(f(n))
```java
for(int i=1; i<=n; i++){
    int j = i;
    j--;
}
```
第一行执行1次
第二行执行n次
第三行执行n次
所以总时间是1+n+n=2n+1
既f(n)=2n+1, 当n无线大时,常数1和系数2可忽略, 则有:
T(n) = O(f(n)) = O(2n+1) = O(n)


常见的时间复杂度量级有:
1. 常数阶O(1)
只要没有循环,比如hashMap查询
2. 线性阶O(n)
3. 对数阶O(logN)

循环内, i每次*2倍, 所以每次循环都是双倍, 假设运行了x次循环, 则2^x=n, 所以x = log_2^n
```java
int i = 1;
while(i<n){
    i = i * 2;
}
```
4. 线性对数阶O(nlogN)
```java
for(int j=1; j<n; j++){
    for(int i=1; i<n; i++){
        i = i * 2;
    }
}
```
5. 平方阶O(n²)

```java
for(int j=1; j<n; j++){
    for(int i=1; i<n; i++){
        i;
    }
}
```
6. 立方阶O(n³)
7. K次方阶O(n^k)
8. 指数阶(2^n)
