# this

this指的是本"对象"

如果函数属于全局,则`this`代表的是全局对象
```js
var x = myFunction();            // x 将成为 window 对象

function myFunction() {
   return this;
}
```

如果函数属于某个对象,则`this`代表的是该对象

```js
function Person(name, age) {
    this.name = first;
    this.age = age;
}
var father = new Person("wang", 18);
```
