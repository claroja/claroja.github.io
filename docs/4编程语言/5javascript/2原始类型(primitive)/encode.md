# encode


将字符串(unicode, utf8)转换为二进制
```js
let uint8array: Uint8Array= new TextEncoder().encode("王");
console.log(uint8array)  // Uint8Array(3) [ 231, 142, 139 ]  "王"用了三个字节来存储
let string: String = new TextDecoder().decode(uint8array);
console.log(string)  //
```
