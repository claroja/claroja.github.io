## if
1. if
```c
if(条件){
	语句;
}
```

2. if-else
```c
if(条件){
	语句;
}else{
	语句;
	}
```

3. else if
```c
if(条件1){
	语句;
}else if(条件2){
	语句;
}else{
	语句;
}
```
## switch-case
```c
switch(表达式){
    case 整型常量1:
    	语句;
       break; /* 可选的 */
    case 整型常量2:
       语句;
       break; /* 可选的 */
    default : /* 可选的 */
       语句;
}
```

比较|!=|不等
逻辑|!|非
逻辑|&&|与
逻辑|\|\||或
