Variance和CoVariance

Variance

Variance measures the variation of a single random variable(单个随机变量的离散程度)
variance的formula:

  1. 仅使用sample data, 只能获得sample mean, 使用n-1做为分子

σx2=1n−1∑i=1n(xi−xˉ)2\sigma^2_x = \frac{1}{n-1} \sum^{n}_{i=1}(x_i - \bar{x})^2 σx2​=n−11​i=1∑n​(xi​−xˉ)2

  1. 使用population data, 知道population mean, 使用n作为分子

σx2=1n∑i=1n(xi−xˉ)2\sigma^2_x = \frac{1}{n} \sum^{n}_{i=1}(x_i - \bar{x})^2 σx2​=n1​i=1∑n​(xi​−xˉ)2

CoVariance

covariance is a measure of how much two random variables vary together(两个随机变量的相似程度)
covariance的formula:

σ(x,y)=1n−1∑i=1n(xi−xˉ)(yi−yˉ)\sigma(x, y) = \frac{1}{n-1} \sum^{n}_{i=1}{(x_i-\bar{x})(y_i-\bar{y})} σ(x,y)=n−11​i=1∑n​(xi​−xˉ)(yi​−yˉ​)

σ(x,y)=1n∑i=1n(xi−xˉ)(yi−yˉ)\sigma(x, y) = \frac{1}{n} \sum^{n}_{i=1}{(x_i-\bar{x})(y_i-\bar{y})} σ(x,y)=n1​i=1∑n​(xi​−xˉ)(yi​−yˉ​)

Variance和CoVariance的区别就是: CoVariance使用了两个变量的的mean.
variance σx2\sigma_x^2σx2​可以写成它自己的协方差σ(x,x)\sigma(x, x)σ(x,x)

Positive, Negative, and Zero States of The Covariance

当xi−xˉx_i-\bar{x}xi​−xˉ和yi−yˉy_i-\bar{y}yi​−yˉ​同时为negative或positive时, 他们的乘积就是正向的, 如果最后加和为positive, 表明XXX和YYY方向相同. 同理可知为negative的情况.
而当Covariance为0, 说明两个随机变量并没有相关关系.如下图:

另外一个场景是, covariance为0, 而每个随机变量的variance不同.

因为covariance没有归一化到[-1,1]中, 因此不能说从covariance越大两者的关系就越强

CoVariance Matrix

Ci,j=σ(xi,xj)C_{i,j} = \sigma(x_i, x_j)Ci,j​=σ(xi​,xj​) where C∈Rd×dC \in \mathbb{R}^{d \times d}C∈Rd×d ddd代表了维度或者说随机变量的数量(比如身高,体重,年龄等).
covariance matrix是symmetric, σ(xi,xj)=σ(xj,xi)\sigma(x_i, x_j) = \sigma(x_j, x_i)σ(xi​,xj​)=σ(xj​,xi​).
对角线(diagonal)上的元素是variances, 对角线之外的元素是covariances.
CoVariance Matrix的formula是:
C=1n−1∑i=1n(Xi−Xˉ)(Xi−Xˉ)TC = \frac{1}{n-1} \sum^{n}_{i=1}{(X_i-\bar{X})(X_i-\bar{X})^T}C=n−11​∑i=1n​(Xi​−Xˉ)(Xi​−Xˉ)T

定义一个二维的CoVariance Matrix

C=(σ(x,x)σ(x,y)σ(y,x)σ(y,y))C = \left( \begin{array}{ccc} \sigma(x, x) & \sigma(x, y) \\ \sigma(y, x) & \sigma(y, y) \end{array} \right)C=(σ(x,x)σ(y,x)​σ(x,y)σ(y,y)​)

我们来生成xˉ\bar{x}xˉ=yˉ\bar{y}yˉ​=0, σx2=σy2=1\sigma^2_x = \sigma^2_y = 1σx2​=σy2​=1

import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline

plt.style.use('ggplot')
plt.rcParams['figure.figsize'] = (12, 8)


# Normal distributed x and y vector with mean 0 and standard deviation 1
x = np.random.normal(0, 1, 500)
y = np.random.normal(0, 1, 500)
X = np.vstack((x, y)).T

plt.scatter(X[:, 0], X[:, 1])
plt.title('Generated Data')
plt.axis('equal')


这种情况意味着xxx和yyy是独立的, covariance matrix CCC是

C=(σx200σy2)C = \left( \begin{array}{ccc} \sigma_x^2 & 0 \\ 0 & \sigma_y^2 \end{array} \right)C=(σx2​0​0σy2​​)

我们来计算验证一下:

# Covariance
def cov(x, y):
    xbar, ybar = x.mean(), y.mean()
    return np.sum((x - xbar)*(y - ybar))/(len(x) - 1)

# Covariance matrix
def cov_mat(X):
    return np.array([[cov(X[0], X[0]), cov(X[0], X[1])], \
                     [cov(X[1], X[0]), cov(X[1], X[1])]])

# Calculate covariance matrix 
cov_mat(X.T) # (or with np.cov(X.T))

# array([[ 1.008072  , -0.01495206],
#        [-0.01495206,  0.92558318]])

Linear Transformations

scaling

使用下面的scaling matrix对CCC进行转换(矩阵乘积):

S=(sx00sy)S = \left( \begin{array}{ccc} s_x & 0 \\ 0 & s_y \end{array} \right)S=(sx​0​0sy​​)

既Y=SXY=SXY=SX

矩阵相乘后我们期望得到以下的结果,既:

C=((sxσx)200(syσy)2)C = \left( \begin{array}{ccc} (s_x\sigma_x)^2 & 0 \\ 0 & (s_y\sigma_y)^2 \end{array} \right)C=((sx​σx​)20​0(sy​σy​)2​)

# Center the matrix at the origin
X = X - np.mean(X, 0)

# Scaling matrix
sx, sy = 0.7, 3.4
Scale = np.array([[sx, 0], [0, sy]])

# Apply scaling matrix to X
Y = X.dot(Scale)

plt.scatter(Y[:, 0], Y[:, 1])
plt.title('Transformed Data')
plt.axis('equal')

# Calculate covariance matrix
cov_mat(Y.T)

结果正式我们所期望的(sxσx)2=(0.7∗1)2=0.49,(syσy)2=(3.4∗1)2=11.56(s_x\sigma_x)^2=(0.7*1)^2=0.49,(s_y\sigma_y)^2=(3.4*1)^2=11.56(sx​σx​)2=(0.7∗1)2=0.49,(sy​σy​)2=(3.4∗1)2=11.56

linear transformation

使用rotation matrix RRR和scaling matrix SSS进行转换:
T=RST=RST=RS
其中RRRmatrix为:

R=(cos(θ)−sin(θ)sin(θ)cos(θ))R = \left( \begin{array}{ccc} cos(\theta) & -sin(\theta) \\ sin(\theta) & cos(\theta) \end{array} \right)R=(cos(θ)sin(θ)​−sin(θ)cos(θ)​)

其中θ\thetaθ是旋转的角度, 使用如下公式进行旋转:
Y=TX or Y=RSXY=TX\ or\ Y=RSXY=TX or Y=RSX

# Scaling matrix
sx, sy = 0.7, 3.4
Scale = np.array([[sx, 0], [0, sy]])

# Rotation matrix
theta = 0.77*np.pi
c, s = np.cos(theta), np.sin(theta)
Rot = np.array([[c, -s], [s, c]])

# Transformation matrix
T = Scale.dot(Rot)

# Apply transformation matrix to X
Y = X.dot(T)

plt.scatter(Y[:, 0], Y[:, 1])
plt.title('Transformed Data')
plt.axis('equal')

# Calculate covariance matrix
cov_mat(Y.T)

# array([[ 4.94072998, -4.93536067],
#        [-4.93536067,  5.99552455]])

Eigen Decomposition

Eigen Decomposition是linear transformation和covariance matrix之间的关联.
An eigenvector is a vector whose direction remains unchanged when a linear transformation is applied to it.
Av=λvAv=\lambda vAv=λv
vvv是 eigenvector, λ\lambdaλ是eigenvalue

如果我们将eigenvectors,按列放入Matrix VVV,所有的eigenvalues作为matrix LLL的diagonal元素, 则covariance matrix CCC可以写成:

CV=VLCV = VL CV=VL

变换后可得:

C=VLV−1C = VLV^{-1} C=VLV−1

C = cov_mat(Y.T)
eVe, eVa = np.linalg.eig(C)

plt.scatter(Y[:, 0], Y[:, 1])
for e, v in zip(eVe, eVa.T):
    plt.plot([0, 3*np.sqrt(e)*v[0]], [0, 3*np.sqrt(e)*v[1]], 'k-', lw=2)
plt.title('Transformed Data')
plt.axis('equal')
