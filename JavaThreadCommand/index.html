查看Process/Thread方法

win

查看进程

  • tasklist | findstr java
  • jps
    杀死进程
  • taskkill /F /PID杀死进程

Linux

  • ps -fe | grep java 查看所有java进程
  • ps -fT -p <PID> 查看某个进程的所有线程
  • top 动态查看所有进程
  • top -H -p <PID> 动态查看某进程的所有线程
  • jps
  • jstack <PID> 更详细的信息
